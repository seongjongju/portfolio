프롬헤라스 웹 사이트

프로젝트 개요

프롬헤라스 웹 사이트를 클론 코딩하며 독특한 가로 스크롤 방식의 웹 사이트를 구현하였습니다.

프롬헤라스는 가로로 자연스럽게 스크롤되며, 사용자에게 색다른 인터랙티브 경험과 즐거움을 제공합니다. 이러한 작업을 통해 사용자 경험(UX)과 시각적 요소가 조화롭게 어우러진 웹 디자인을 완성하였습니다.

작업 기간

· 퍼블리싱 1주

작업 범위

· 메인페이지

기술 스택

· HTML

· CSS

· JavaScript

· jquery

기여도

· 퍼블리싱 100%

주요기능

메인페이지
· 실제 사이트와 유사한 레이아웃을 구현하였으며, 가로로 이동하면서 다양한 애니메이션 효과를 통해 사용자에게 즐거움을 제공합니다.

개선 및 배운점

즐거운 애니메이션이 사용자 경험을 어떻게 향상시키는지를 깨달았습니다. 사용자에게 직관적이고 재미있는 환경을 제공하는 것이 중요함을 배웠습니다.

다양한 애니메이션 효과가 웹사이트의 매력을 높일 수 있다는 점을 이해하게 되었고, 이를 통해 사용자의 몰입도를 높이는 방법을 익혔습니다.
