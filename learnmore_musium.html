국립현대미술관 반응형 웹 사이트

프로젝트 개요

국립현대미술관의 웹사이트를 클론 코딩하여, 실제 사이트의 구조와 디자인을 분석하고 구현한 작업입니다.

데스크탑부터 모바일까지 다양한 디바이스 환경에서 원활하게 실행될 수 있도록 반응형 웹 작업을 포함하여, 사용자 경험을 고려한 최적화된 설계를 반영하였습니다.

작업 기간

· 퍼블리싱 2주

작업 범위

· 메인페이지

· 관람참여 > 서울

· 관람참여 > 과천

· 소장품 > 소장품 검색

기술 스택

· HTML

· CSS

· JavaScript

· jquery

· Responsive Design

기여도

· 퍼블리싱 100%

주요기능

메인페이지
· 실제 사이트와 유사한 레이아웃 구현, 주요 전시 및 소식 배너 표시하였습니다.

관람참여
· 층별안내, 오신는 길은 탭메뉴, 편의시설과 문화시설은 스와이퍼를 이용하여 콘텐츠를 적절히 배치하였습니다.

소장품
· 필터링 및 정렬 기능 추가하였습니다.

반응형 디자인
· 다양한 기기에서 최적화된 사용자 경험 제공합니다.

반응형 GNB
· 모바일 환경에서 사용자가 편리하게 이용할 수 있도록 오른쪽에서 슬라이드 되는 기능을 구현하였으며, 하위 메뉴는 아코디언 메뉴로 구성하였습니다.

개선 및 배운점

클론 코딩 과정에서 HTML과 CSS 구조의 중요성, 그리고 자바스크립트를 통한 동적 요소 구현의 필요성을 깊게 깨달았습니다.

중복 클래스를 활용하여 효율적인 퍼블리싱 방법을 익혔습니다.

모바일 환경을 고려한 데스크탑 환경 구현의 중요성을 체감하게 되었습니다.
