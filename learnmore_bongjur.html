Bonjour.Paris Eshop 웹 사이트

프로젝트 개요

Bonjour.Paris Eshop 웹 사이트를 클론 코딩하며 독특하고 도전적인 디자인 요소를 분석하고 구현하였습니다.

쇼핑몰의 핵심 기능을 구현하면서, 직관적이면서도 시각적으로 매력적인 UI/UX를 재현하는 데 주력하였습니다.

작업 기간

· 퍼블리싱 2주

작업 범위

· 메인페이지

· ÉDITIONS LIMITÉES

· ÉDITIONS LIMITÉES 개별 페이지

· Sweatshirts

기술 스택

· HTML

· CSS

· JavaScript

· jquery

기여도

· 퍼블리싱 100%

주요기능

메인페이지
· 실제 사이트와 유사한 레이아웃 구현하였습니다.

GNB
· 독특한 레이아웃을 구현하였으며, 마우스를 올리면 해당 이미지가 등장하는 효과를 추가하였습니다.

ÉDITIONS LIMITÉES
· 가격별로 정렬되는 기능을 구현하였습니다.

ÉDITIONS LIMITÉES 개별 페이지
· 쇼핑몰의 주요 기능인 색상 및 사이즈 선택과 수량 선택에 따른 총 가격 계산 기능을 구현하였습니다.

개선 및 배운점

GNB에서 마우스 오버 효과를 사용함으로써 사용자와의 상호작용을 통해 디자인의 매력을 높일 수 있다는 것을 깨달았습니다.

쇼핑몰에서 사용자 편의성을 높이는 다양한 기능(가격별 정렬, 선택 및 계산 기능)의 중요성을 배웠습니다.
